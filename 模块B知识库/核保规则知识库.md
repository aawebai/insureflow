# 保险核保规则知识库

## 一、职业风险等级与费率调整

### 1级风险职业 — 加费0%
属于最低风险等级，按标准费率承保，无职业加费。

具体职业包括：程序员、软件工程师、前端开发工程师、后端开发工程师、数据分析师、文员、行政专员、办公室职员、行政助理、人事专员、教师、幼儿园老师、小学老师、中学老师、大学教授、学生、退休人员、会计、注册会计师、律师、法务、设计师、平面设计师、UI设计师、产品经理、运营专员、新媒体运营、客服人员、电话客服、在线客服、人力资源、人力资源经理、财务人员、出纳、审计师、图书管理员、编辑、记者（非战地）、翻译、心理咨询师、营养师、研究员、实验室人员。

这些职业的共同特征是：在室内固定场所工作，不接触危险设备或环境，工作强度和压力相对可控。

### 2级风险职业 — 加费0%
属于较低风险等级，按标准费率承保，无职业加费。

具体职业包括：销售员、房产中介、保险代理人、商店店员、超市收银员、快递站内勤、物流客服、物业管理员、保安（室内/小区）、家政人员、保姆、月嫂、出租车司机、网约车司机、代驾司机、外卖骑手、快递员（小件/非重物）、快递配送员、邮政投递员、餐厅服务员、酒店前台、导游、驾校教练、社区网格员、城管协管员。

这些职业的共同特征是：偶尔需要外出或轻度体力劳动，但整体风险可控，不涉及高空、深水、有毒有害环境。

### 3级风险职业 — 加费10%
属于中等风险等级，需要在标准费率基础上加费10%。

具体职业包括：建筑工人、装修工人、泥瓦工、木工、油漆工、水电工、管道工、电焊工、气焊工、机械操作工、车床工、铣工、冲压工、化工厂操作工、化工厂巡检员、矿工（地面作业）、采石场地面工、货车司机、长途货运司机、油罐车司机、混凝土搅拌车司机、消防员（非一线/行政岗）、警察（户籍/内勤）、交警（非路面）、城管执法队员、农民、农业机械操作员、渔民（近海）、养殖场工人、屠宰场工人、垃圾清运工、高空清洁工（低层）。

这些职业的共同特征是：从事一定体力劳动，或在有一定风险的场所工作，发生意外的概率高于普通室内工作者。

### 4级风险职业 — 加费20%-30%
属于高风险等级，需要在标准费率基础上加费20%至30%。

具体职业包括：高空作业人员、外墙清洗工（高层）、塔吊司机、起重机操作员、架子工、钢筋工（高空绑扎）、爆破工、爆破员、矿工（井下作业）、煤矿井下工人、深海渔民、远洋渔船船员、消防员（一线灭火）、森林消防员、刑警、缉毒警察、特警、防暴警察、武装押运员、职业运动员（接触性运动）、拳击运动员、武术教练、马术运动员、赛车维修技师、直升机飞行员（非航线）、伐木工、高压电塔维修工。

### 5级风险职业 — 加费50%-100%
属于极高风险等级，需要在标准费率基础上加费50%至100%。

具体职业包括：试飞员、飞机试飞员、深海潜水员、商业潜水员、拆弹专家、排爆手、战地记者、远洋船员、远洋货轮海员、私人保镖、赛车手、摩托车赛车手、竞技跳伞运动员、高山向导、登山向导、火山学家（野外作业）、北极/南极科考队员。

### 6级风险职业 — 拒保
属于不可承保风险等级，直接拒保。

具体职业包括：雇佣兵、私人军事承包商（战区）、任何非法职业、无安全保障的非法矿工、没有任何劳动保护措施的高危作业人员。

## 二、各险种核保规则与要点

### 重疾险
重疾险以被保人罹患合同约定的重大疾病为给付条件。核保重点关注被保人的年龄、家族病史、现有健康状况和生活习惯。

基准费率（元/年/10万保额）：
- 男性：18岁档800元、30岁档1200元、40岁档2200元、50岁档4200元、60岁档7800元
- 女性：18岁档720元、30岁档1080元、40岁档1980元、50岁档3780元、60岁档7000元

常见加费因素：高血压、糖尿病、甲状腺结节、乳腺结节、乙肝携带、超重/肥胖、吸烟、饮酒过量。

常见拒保情形：已确诊恶性肿瘤、糖尿病合并并发症、严重心脑血管疾病（心肌梗死、脑卒中史）、严重高血压（≥180/110mmHg）、肝硬化失代偿期、尿毒症/慢性肾病4-5期、系统性红斑狼疮活动期、再生障碍性贫血。

延期处理情形：近期发现的未明确诊断的健康异常、正在治疗中且预后不明的疾病、妊娠期（分娩后可投保）。

### 医疗险
医疗险以实际发生的医疗费用为报销依据。核保重点关注被保人的既往病史、当前健康状况和职业风险。

基准费率（元/年/10万保额）：
- 男性：18岁档350元、30岁档500元、40岁档900元、50岁档1800元、60岁档3200元
- 女性：18岁档320元、30岁档460元、40岁档820元、50岁档1650元、60岁档2900元

常见加费因素：任何在治疗中的慢性病、BMI异常、吸烟、有住院史（近2年内）、有门诊频繁就诊记录。

常见拒保情形：恶性肿瘤病史（含已治愈）、严重心脑血管疾病、肝硬化、慢性肾病3期及以上、器官移植术后、严重糖尿病伴并发症、精神类疾病（精神分裂症、双相障碍）。

延期处理情形：近期手术未满6个月、体检异常待复查、急性疾病恢复期。

### 定期寿险
定期寿险以被保人在保险期间内身故或全残为给付条件。核保重点关注被保人的年龄、职业风险和健康状况。

基准费率（元/年/10万保额）：
- 男性：18岁档400元、30岁档650元、40岁档1200元、50岁档2800元、60岁档5500元
- 女性：18岁档360元、30岁档580元、40岁档1080元、50岁档2500元、60岁档4800元

常见加费因素：高危职业、多项健康异常、重度吸烟、BMI严重超标。

常见拒保情形：预期寿命明显缩短的严重疾病、确诊HIV/AIDS、终末期疾病、5级及以上职业风险（部分公司）。

### 意外险
意外险以意外伤害导致的身故、残疾或医疗费用为给付条件。核保核心是职业风险等级，健康状况的影响相对较小。

基准费率（元/年/10万保额）：
- 男女统一：18岁档120元、30岁档150元、40岁档200元、50岁档300元、60岁档500元
- 意外险男女费率相同，不做性别区分

常见拒保情形：6级职业风险、从事高危运动/活动（跳伞、攀岩、潜水、赛车等）且不购买专项保险者。

特别注意：意外险对职业风险极为敏感。建筑工人、外卖骑手等3级以上职业需按对应风险等级加费或购买专项意外险。

## 三、健康状况核保规则与加费标准

### 高血压
血压单位：mmHg。高血压是核保中最重要的健康风险因素之一。

- 血压正常（<120/<80）：标体承保，不加费
- 正常高值（120-139/80-89）：标体承保，不加费
- 1级高血压（140-159/90-99）：加费20%
- 1级高血压（服药控制，血压稳定在140/90以下）：加费20%
- 2级高血压（160-179/100-109）：加费40%
- 3级高血压（≥180/≥110）：加费80%或直接拒保
- 高血压合并糖尿病：加费50%或拒保（视具体数值和目标器官损害情况）
- 高血压合并冠心病/脑卒中史：拒保
- 高血压合并肾功能异常（肌酐升高）：拒保或加费80%

### 糖尿病
- 空腹血糖正常（<6.1mmol/L），糖化血红蛋白正常：标体
- 空腹血糖受损（6.1-7.0mmol/L），糖尿病前期：加费20%
- 2型糖尿病（糖化血红蛋白<7.0%，无并发症，口服药控制良好）：加费50%
- 2型糖尿病（糖化血红蛋白7.0%-8.5%，有轻微并发症）：加费80%
- 2型糖尿病合并心血管并发症：拒保
- 2型糖尿病合并肾病（尿蛋白阳性）：拒保
- 2型糖尿病合并视网膜病变：拒保
- 1型糖尿病：拒保
- 妊娠期糖尿病（产后血糖恢复正常）：标体

### 高血脂/高胆固醇
单位：mmol/L。TC=总胆固醇，LDL=低密度脂蛋白，TG=甘油三酯。

- 血脂正常范围：标体
- 轻度升高（TC 5.2-6.2 或 LDL 3.4-4.1 或 TG 1.7-2.3）：加费10%
- 中度升高（TC 6.2-7.2 或 LDL 4.1-4.9 或 TG 2.3-5.6）：加费20%
- 重度升高（TC >7.2 或 LDL >4.9 或 TG >5.6）：加费40%
- 高血脂合并冠心病/脑卒中：拒保

### 甲状腺结节
采用TI-RADS分级系统：
- TI-RADS 1级（正常甲状腺）：标体
- TI-RADS 2级（良性）：标体
- TI-RADS 3级（可能良性，恶性风险<5%）：加费20%
- TI-RADS 4A级（低度怀疑恶性）：加费50%
- TI-RADS 4B级（中度怀疑恶性）：延期至明确诊断
- TI-RADS 4C级（高度怀疑恶性）：延期或拒保
- TI-RADS 5级（典型恶性征象）：拒保

### 乳腺结节
采用BI-RADS分级系统：
- BI-RADS 1级（正常乳腺）：标体
- BI-RADS 2级（良性）：标体
- BI-RADS 3级（可能良性，恶性风险<2%）：加费15%
- BI-RADS 4A级（低度怀疑恶性）：加费40%
- BI-RADS 4B级及以上：延期或拒保
- BI-RADS 5级及以上：拒保

### 脂肪肝
- 轻度脂肪肝（无肝功能异常，ALT/AST正常）：加费10%
- 中度脂肪肝（ALT轻度升高，<正常上限2倍）：加费25%
- 重度脂肪肝或合并肝功能明显异常（ALT>正常上限2倍）：加费50%
- 脂肪肝合并糖尿病或代谢综合征：加费80%或拒保
- 酒精性脂肪肝：按酒精依赖规则处理
- 非酒精性脂肪肝合并肝纤维化≥F2期：拒保

### 乙肝
- 乙肝表面抗原阴性（未感染）：标体
- 乙肝表面抗体阳性（有免疫力）：标体
- 乙肝小三阳（HBsAg+, HBeAb+, HBcAb+），肝功能正常，HBV-DNA阴性：加费30%
- 乙肝小三阳，肝功能正常，HBV-DNA阳性但低于检测下限：加费40%
- 乙肝大三阳（HBsAg+, HBeAg+, HBcAb+），肝功能正常，HBV-DNA阴性/低水平：加费50%
- 乙肝大三阳，肝功能异常：延期或拒保
- 乙肝肝硬化：拒保
- 肝癌家族史合并乙肝：拒保或加费80%
- 丙肝抗体阳性，RNA阳性：拒保
- 丙肝已治愈（RNA持续阴性>6个月）：加费30%

### 肾脏疾病
- 单纯性肾囊肿（<3cm，无症状）：标体
- 肾囊肿（3-5cm）：加费15%
- 肾囊肿（>5cm或多发）：加费30%
- 肾结石（<1cm，无症状，无肾积水）：标体
- 肾结石（1-2cm或有轻度肾积水）：加费25%
- 肾结石（>2cm或反复发作）：加费50%
- 慢性肾炎/慢性肾病1-2期：加费50%-80%
- 慢性肾病3期及以上：拒保
- 尿毒症/透析：拒保

### 其他常见异常
- 轻度贫血（Hb 90-110g/L）：加费15%
- 中度贫血（Hb 60-90g/L）：延期
- 重度贫血（Hb <60g/L）：拒保
- 子宫肌瘤（<5cm，单个，无症状）：标体
- 子宫肌瘤（>5cm或多发或有症状）：加费20%
- 胆囊息肉（<1cm，单发）：标体
- 胆囊息肉（≥1cm或多发）：加费20%
- 痛风（偶发，无痛风石，尿酸控制良好）：加费15%
- 痛风（频发或有痛风石形成）：加费40%
- 哮喘（轻中度，控制良好，无需长期用药）：加费20%
- 哮喘（重度或控制不佳）：加费50%或拒保
- 抑郁症（轻度，稳定期>2年，无需药物维持）：加费25%
- 抑郁症（重度或曾住院）：拒保
- 甲状腺功能亢进（已治愈/稳定>1年）：标体
- 甲状腺功能减退（稳定服药，甲功正常）：标体加费10%

## 四、BMI体重指数核保规则

BMI计算公式：体重（kg）÷ 身高（m）²

- BMI < 16.0：重度营养不良，加费50%，需排查消耗性疾病
- BMI 16.0-18.4：体重过轻，加费10%，需排查慢性消耗性疾病
- BMI 18.5-23.9：正常范围，不加费
- BMI 24.0-27.9：超重，加费15%
- BMI 28.0-31.9：肥胖，加费30%
- BMI 32.0-39.9：重度肥胖，加费50%
- BMI ≥ 40.0：病态肥胖，拒保

附加规则：
- BMI超标合并高血压、高血脂、高血糖中任意一项：在BMI加费基础上额外加费10%
- BMI超标合并两项以上代谢异常：加费50%或拒保
- 体重突然大幅变化（半年内变化>10%）：需排查原因后再评估
- 腹型肥胖（腰围男>90cm/女>85cm）：即使BMI正常也需加费10%

## 五、吸烟核保规则与加费标准

- 不吸烟者：不加费，吸烟风险系数为1.0
- 轻度吸烟者（每天<10支，吸烟史<5年）：加费15%，吸烟风险系数为1.15
- 中度吸烟者（每天10-20支，吸烟史5-15年）：加费15%，吸烟风险系数为1.15
- 重度吸烟者（每天≥20支，吸烟史≥15年，或吸烟指数≥400）：加费30%，吸烟风险系数为1.30
- 已戒烟满1年以上：可按不吸烟处理，不加费
- 已戒烟6个月至1年：仍加费15%，满1年后可申请复核
- 已戒烟不足6个月：仍按原吸烟量加费
- 吸烟合并COPD/肺气肿：拒保
- 吸烟合并肺癌家族史：加费50%

备注：吸烟指数 = 每天吸烟支数 × 吸烟年数。吸烟指数 > 400 为肺癌高危人群。

## 六、年龄风险系数

年龄是核保定价中最基础的参数之一，不同年龄段对应不同的风险系数。

- 年龄 ≤ 17岁：风险系数 0.8（未成年人风险较低，保费有优惠）
- 年龄 18-40岁：风险系数 1.0（标准年龄段，按基准费率计算）
- 年龄 41-55岁：风险系数 1.3（中年风险上升，保费上浮30%）
- 年龄 56-65岁：风险系数 1.6（中老年阶段，保费上浮60%）
- 年龄 > 65岁：风险系数 2.0（老年阶段，保费翻倍）

附加说明：
- 年龄越小投保越划算，保费更低且核保更容易通过
- 超过60岁投保，多数险种将限制最高保额
- 超过65岁投保，可选险种将大幅减少
- 超过70岁投保，通常仅能购买老年专属产品或意外险

## 七、综合核保决策标准

### 标准体（正常承保）
认定条件：
- 综合风险系数 ≤ 1.0
- 职业风险等级 ≤ 2级
- 健康风险评估为"标准"或"无异常"
- 无吸烟或已戒烟满1年
- BMI在正常范围内（18.5-23.9）

承保方案：按标准费率正常承保，无附加条件，无除外责任。

### 次标准体（加费/限额承保）
认定条件：
- 综合风险系数 1.01-1.5
- 或存在单一可逆/可控的健康风险
- 职业风险等级为3级

承保方案：给出具体的加费比例或保额限额方案，客户接受后即可承保。

### 延期体（延期处理）
认定条件：
- 健康异常暂无法明确评估风险程度
- 预计3-6个月内病情可改善或可明确诊断
- 近期有手术或住院史（6个月内）
- 体检异常待复查且不影响紧急投保需求

延期方案：给出延期时长（3个月/6个月/1年）和后续投保建议，明确说明延期的原因和好转后可投保的条件。

### 拒保体（拒绝承保）
认定条件：
- 综合风险系数 > 1.5
- 存在明确的拒保指征（如已确诊恶性肿瘤、严重心脑血管疾病等）
- 职业风险等级为6级
- 多项严重健康异常叠加

拒保方案：给出具体的拒保理由，建议合适的替代险种（如意外险、防癌险等核保门槛较低的产品），或建议转社保/惠民保等政策性保险。

## 八、常见组合场景核保参考

1. **年轻健康 + 低风险职业 + 标准BMI → 标准体**：28岁程序员，不吸烟，BMI 22.9，买重疾险 → 综合风险系数1.0，标准体正常承保。

2. **中年 + 高风险职业 + 多项健康异常 → 拒保体**：45岁建筑工人，吸烟10年，高血压服药控制，BMI 29.4（肥胖），买医疗险 → 综合风险系数约2.6（>1.5），建议拒保或转投防癌险/意外险。

3. **年轻女性 + 低风险职业 + 健康 → 标准体**：22岁女性文员，身体健康不吸烟，买定期寿险 → 综合风险系数1.0，标准体正常承保。

4. **高龄 + 多项慢性病 → 拒保体**：68岁退休人员，2型糖尿病合并高血压，买重疾险 → 综合风险系数远超2.0，直接拒保，建议转投老年意外险或惠民保。

5. **信息不全 → 默认值处理**：年龄未知默认按30岁计算，性别未知默认男性费率，职业未知按1级风险处理，健康未知按标准体处理，但需提示用户补充信息以获取精确报价。

6. **外卖骑手 + 意外险 → 标准体**：35岁外卖骑手，买意外险（保额较低时），职业风险等级2级，健康无异常 → 综合风险系数1.0，标准体承保。但高保额或含伤残责任时需加费。

7. **未成年 + 医疗险 → 优惠承保**：17岁学生，买医疗险，健康 → 年龄系数0.8，综合风险系数0.8，保费为基准的80%。

8. **意外险男女同价**：意外险费率不区分性别，30岁男性和30岁女性意外险基准费率均为150元/年/10万保额。
