# 保险产品费率定价指导原则

本文档提供保险产品定价的核心方法论、费率区间参考和定价因子权重，供产品配置工作流进行费率框架设计。

---

## 一、重疾险定价体系

### 1.1 定价因子权重

| 定价因子 | 权重 | 说明 |
|----------|:---:|------|
| 年龄 | 40% | 最重要的定价因子，费率随年龄指数型增长。每增加1岁，费率上浮约3-5%（40岁后加速） |
| 性别 | 5% | 女性费率略低于男性（约低5-10%），因女性平均寿命更长且部分重疾发生率较低 |
| 保障期限（定期vs终身） | 15% | 定期（至70岁）费率约为终身的50-60%。保障期越长，费率越高 |
| 病种数量 | 10% | 每增加10种重疾，费率上浮约3-5%。病种增加到一定量后边际影响递减 |
| 多次赔付（含分组） | 10% | 每增加一次重疾赔付次数，费率上浮约15-20%。分组赔付比不分组低约10% |
| 轻症/中症覆盖 | 8% | 轻症数量越多、给付比例越高、次数越多，费率越高 |
| 豁免条款 | 5% | 轻症豁免使费率上浮约3-5%。投保人豁免（附加）需单独定价 |
| 等待期 | 3% | 90天等待期 vs 180天等待期，费率差异约2-3% |
| 渠道费用 | 4% | 包含销售佣金、运营费用分摊 |

### 1.2 费率区间参考（元/年/10万保额）

| 年龄段 | 男性-定期(至70岁) | 男性-终身 | 女性-定期(至70岁) | 女性-终身 |
|:---:|:---:|:---:|:---:|:---:|
| 0-17岁 | 80-120 | 200-300 | 70-110 | 180-270 |
| 18-30岁 | 600-1,000 | 1,000-1,600 | 540-900 | 900-1,440 |
| 31-40岁 | 1,200-1,800 | 2,000-3,000 | 1,080-1,620 | 1,800-2,700 |
| 41-50岁 | 2,200-3,200 | 3,800-5,200 | 1,980-2,880 | 3,420-4,680 |
| 51-55岁 | 3,200-4,500 | 5,500-7,500 | 2,880-4,050 | 4,950-6,750 |
| 56-60岁 | 4,500-6,000 | 7,500-10,000 | 4,050-5,400 | 6,750-9,000 |

> 说明：以上为含轻症/中症/身故的标准重疾险费率区间。不含身故的"纯重疾"费率约为含身故版的60-70%。

### 1.3 多次赔付加费参考

| 赔付次数 | 相对于单次赔付的费率倍数 |
|:---:|:---:|
| 2次（分组） | 1.15-1.25 |
| 2次（不分组） | 1.25-1.35 |
| 3次（分组） | 1.30-1.45 |
| 3次（不分组） | 1.45-1.60 |
| 恶性肿瘤二次赔付（附加） | 主险费率的1.10-1.20 |

### 1.4 特色责任加费参考

| 特色责任 | 加费幅度 |
|------|:---:|
| 60岁前首次重疾额外给付50%保额 | +12-18% |
| 60岁前首次重疾额外给付80%保额 | +18-25% |
| 前10年首次重疾额外给付50%保额 | +8-12% |
| 运动达标增保额（最高10%） | +3-5% |
| 特定疾病额外给付（如少儿特疾） | +5-10% |
| 中症给付比例从50%提升至60% | +3-5% |
| 轻症给付比例从25%提升至30% | +2-4% |

## 二、医疗险定价体系

### 2.1 定价因子权重

| 定价因子 | 权重 | 说明 |
|----------|:---:|------|
| 年龄 | 50% | 年龄越大，医疗使用频率越高，费率增长明显 |
| 有无社保 | 15% | 有社保版费率约为无社保版的60-70% |
| 保证续保年限 | 10% | 保证续保期越长，费率越高（因锁定长期风险） |
| 免赔额 | 10% | 免赔额越高，费率越低。0免赔版费率约为1万免赔版的1.5-2倍 |
| 保额 | 8% | 一般医疗保额从100万提升到300万，费率增幅不超过10%（边际递减） |
| 特色服务 | 7% | 质子重离子、特药报销、门诊绿通等服务增加运营成本 |

### 2.2 费率区间参考（元/年，有社保版）

| 年龄段 | 一般医疗200万+重疾400万，1万免赔额 | 含0免赔 |
|:---:|:---:|:---:|
| 0-5岁 | 600-1,000 | 1,200-2,000 |
| 6-17岁 | 150-300 | 300-600 |
| 18-25岁 | 150-300 | 300-600 |
| 26-35岁 | 300-500 | 600-1,000 |
| 36-45岁 | 500-800 | 1,000-1,600 |
| 46-55岁 | 1,000-1,800 | 2,000-3,600 |
| 56-60岁 | 1,800-2,800 | 3,600-5,600 |

### 2.3 保证续保对费率的影响

| 保证续保年限 | 费率相对于1年期产品的倍数 |
|:---:|:---:|
| 1年（不保证续保） | 1.00 |
| 3年 | 1.05-1.10 |
| 6年 | 1.10-1.20 |
| 20年 | 1.25-1.40 |

## 三、定期寿险定价体系

### 3.1 定价因子权重

| 定价因子 | 权重 | 说明 |
|----------|:---:|------|
| 年龄 | 45% | 死亡率随年龄增长的核心因子 |
| 性别 | 20% | 男性死亡率显著高于女性，是定期寿险最重要的差异因子 |
| 保障期限 | 15% | 保障期限越长，覆盖的高死亡率年龄段越多，费率越高 |
| 缴费期限 | 10% | 缴费期越长，年缴保费越低（分摊到更多年份） |
| 免责条款数量 | 5% | 免责条款越少（如3条vs7条），费率越高 |
| 保额门槛 | 5% | 高保额（>200万）需要额外核保和体检，费率可能更高 |

### 3.2 费率区间参考（元/年，100万保额，保至60岁，30年缴）

| 年龄段（投保时） | 男性 | 女性 |
|:---:|:---:|:---:|
| 18-25岁 | 600-800 | 350-450 |
| 26-30岁 | 1,000-1,300 | 550-700 |
| 31-35岁 | 1,600-2,000 | 850-1,100 |
| 36-40岁 | 2,500-3,200 | 1,300-1,700 |
| 41-45岁 | 4,000-5,000 | 2,100-2,700 |
| 46-50岁 | 6,000-7,800 | 3,200-4,200 |

### 3.3 保障期限对费率的影响（30岁男性，100万保额，30年缴）

| 保障期限 | 年缴保费 | 相对于保至60岁的倍数 |
|:---:|:---:|:---:|
| 10年 | 350-450 | 0.35 |
| 20年 | 600-800 | 0.60 |
| 30年 | 1,000-1,300 | 1.00 |
| 至60岁 | 1,000-1,300 | 1.00 |
| 至65岁 | 1,400-1,800 | 1.40 |
| 至70岁 | 1,900-2,500 | 1.90 |

## 四、意外险定价体系

### 4.1 定价特点
- 意外险定价不区分性别（男性和女性费率相同），这在人身险中非常特殊。
- 主要原因：意外发生率主要由职业风险和环境风险决定，性别差异不显著。
- 定价核心因子是职业风险等级，健康因素影响极小。

### 4.2 费率区间参考（元/年，10万意外身故保额）

| 职业风险等级 | 年缴保费/10万保额 | 费率倍数 |
|:---:|:---:|:---:|
| 1级（室内办公） | 8-15 | 1.0 |
| 2级（轻度外勤） | 15-25 | 1.5-2.0 |
| 3级（体力劳动） | 25-45 | 2.5-3.5 |
| 4级（高风险） | 50-100 | 5.0-8.0 |
| 5级（极高风险） | 100-200 | 10.0-15.0 |

### 4.3 附加责任加费

| 附加责任 | 相对于基础意外险的加费幅度 |
|------|:---:|
| 意外医疗（含社保外用药） | +30-50% |
| 猝死责任 | +20-35% |
| 意外住院津贴（150元/天） | +15-25% |
| 交通意外额外赔付（航空500万+轨道100万） | +10-20% |

## 五、跨险种通用定价原则

### 5.1 预定利率约束
- 普通型人身保险预定利率上限：3.0%（2025年标准）。
- 分红型保险预定利率上限：2.5%。
- 万能型保险最低保证利率上限：2.0%。
- 预定利率越低，在同样保障责任下保费越高（保险公司需要更保守地估算投资回报）。

### 5.2 定价不得低于成本
- 保费=纯保费（理赔成本）+附加保费（费用+利润）。
- 纯保费不得低于根据行业经验发生率表计算的净保费。
- 附加保费需覆盖佣金、管理费、税费、风险保证金等。
- 恶性价格竞争（低于成本定价）违反偿付能力监管要求。

### 5.3 性别差异约束
- 各险种费率中的性别差异不得超过30%（即女性费率不得低于男性的70%，男性费率不得高于女性的130%）。
- 意外险通常不区分性别费率。
- 如因精算数据支持需要更大性别差异，需向监管提交专项说明。

### 5.4 高龄加费约束
- 原则上65岁以上被保险人的费率不得超过同产品标准体（30岁）费率的3倍。
- 超过3倍的需提供额外的精算依据。
- 部分险种对高龄投保有最高保额限制。

### 5.5 产品定位与费率策略

| 产品定位 | 费率策略 | 费率目标区间 |
|------|------|:---:|
| "性价比旗舰" | 低于市场中位数5-15% | 市场25分位以下 |
| "均衡全能" | 接近市场中位数 | 市场40-60分位 |
| "高端全能" | 高于市场中位数10-25% | 市场75分位以上 |
| "极简入门" | 远低于市场中位数 | 市场10分位以下（需减少责任） |

### 5.6 产品定位与责任配置建议

| 产品定位 | 建议责任配置 | 参考竞品 |
|------|------|------|
| 年轻客群/性价比 | 定期重疾/保额优先/含轻症豁免/不含身故或身故返保费 | 信泰如意尊（定期版） |
| 家庭支柱/均衡 | 终身重疾/多次赔付/含身故/含轻中症豁免/中症50%给付 | 泰康乐享健康 |
| 高净值/全面保障 | 终身重疾/不分组多次赔/额外给付/含恶性肿瘤二次/高端医疗服务 | 平安盛世福 |
| 老年客群/简化 | 定期重疾至70岁/单次赔付/病种精简/无身故 | —（市场空白，机会点） |

---

*本文档费率数据基于行业经验数据、各公司公开费率表及精算假设推导，实际产品定价需由持牌精算师根据公司自身经验数据和再保安排确定。*
