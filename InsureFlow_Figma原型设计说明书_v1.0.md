# InsureFlow 智保通 — 多Agent核保引擎 原型设计说明书

## 文档信息

| 字段 | 内容 |
|------|------|
| 用途 | Figma原型设计参照文档，含5个核心页面 + 完整交互流程 |
| 原型工具 | Figma |
| 设计规范 | 移动端优先（微信小程序/企业微信侧边栏尺寸） |

---

## 一、设计规范

### 1.1 画板尺寸

| 页面类型 | 尺寸 | 使用场景 |
|----------|------|----------|
| 移动端 | 375 × 812 px | 微信小程序 / H5 |
| PC端 | 1024 × 768 px | 内网系统 / 代理人工作台 |

**MVP优先做移动端**（保险代理人80%在移动端使用）。

### 1.2 色彩系统

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色（品牌蓝） | `#2563EB` | 按钮、链接、标题强调 |
| 成功/标准体 | `#10B981` | 核保通过、标准体标识 |
| 警告/次标准体 | `#F59E0B` | 加费承保、需要注意 |
| 延期/延期体 | `#F97316` | 延期投保 |
| 危险/拒保体 | `#EF4444` | 拒保、严重警告 |
| 中性/信息 | `#6B7280` | 辅助文字、次要信息 |
| 背景色 | `#F9FAFB` | 页面底色 |
| 卡片白 | `#FFFFFF` | 卡片背景 |
| 文字主色 | `#111827` | 正文 |
| 文字辅色 | `#6B7280` | 说明文字、标签 |

### 1.3 字体规范

| 层级 | 字号 | 字重 | 用途 |
|------|:---:|:---:|------|
| H1 | 24px | Bold(700) | 页面标题 |
| H2 | 18px | SemiBold(600) | 区块标题 |
| H3 | 16px | Medium(500) | 卡片标题 |
| Body | 14px | Regular(400) | 正文 |
| Caption | 12px | Regular(400) | 辅助说明 |
| Number | 28px | Bold(700) | 重要数字（保费金额） |

### 1.4 组件库

| 组件 | 规格 | 使用场景 |
|------|------|----------|
| 主按钮 | 高44px，圆角8px，品牌蓝底色白字 | 提交核保 |
| 次按钮 | 高44px，圆角8px，白底蓝字边框 | 重新填写 |
| 输入框 | 高48px，圆角8px，灰边框 | 信息输入 |
| 卡片 | 圆角12px，白底，阴影0px 1px 3px rgba(0,0,0,0.1) | 结果展示 |
| 标签(Tag) | 圆角100px，12px字号 | 风险等级/决策标识 |
| 进度条 | 高6px，圆角3px | 核保进度 |
| 分割线 | 1px, `#E5E7EB` | 内容分隔 |

---

## 二、页面清单（5个核心页面 + 2个辅助页面）

```
页面1: 输入页 ──→ 页面2: 处理中页 ──→ 页面3: 结果页
                      │                     │
                      │                     ├─→ 页面4: 报告详情页
                      │                     └─→ 页面5: 替代推荐页（拒保体）
                      │
                  页面A: 错误提示（异常兜底）
                  页面B: 信息不足引导（边缘场景）
```

### 页面2：处理中页（Loading/进度页）

**布局结构**（375×812px）：

```
┌─────────────────────────────────┐
│  智保通核保                      │
├─────────────────────────────────┤
│                                 │
│                                 │
│         ┌─────────────┐         │  ← 动画图标区域
│         │             │         │     120×120px
│         │  🔍 动画     │         │
│         │  (脉冲放大镜) │         │
│         │             │         │
│         └─────────────┘         │
│                                 │
│       正在评估您的核保申请        │  ← H2, 18px
│                                 │
│  ┌───────────────────────────┐  │
│  │ ● 信息提取        已完成   │  │  ← 步骤进度
│  │ ○ 职业风险评估    进行中   │  │     已完成=绿色●
│  │ ○ 健康风险评估    等待中   │  │     进行中=蓝色○+动画
│  │ ○ 综合定价计算    等待中   │  │     等待中=灰色○
│  │ ○ 生成核保报告    等待中   │  │
│  └───────────────────────────┘  │
│                                 │
│  预计还需 5-8 秒                 │  ← Caption, 灰色
│                                 │
│  ┌───────────────────────────┐  │
│  │  智能核保进度              │  │  ← 进度条, 6px高
│  │  ████████░░░░░░░░  45%    │  │     蓝色#2563EB
│  └───────────────────────────┘  │
│                                 │
│  正在分析：职业风险等级...        │  ← 动态描述文字
│                                 │
└─────────────────────────────────┘
```

**交互说明**：
- 步骤进度条5个步骤依次点亮
- 中间图标持续脉冲动画（放大镜搜索效果）
- 动态描述文字每2秒切换（"正在分析：职业风险等级..."→"正在评估：健康状况..."→"正在计算：综合保费..."）
- 进度条45%→60%→75%→90%→100%平滑推进
- 完成后自动跳转结果页（不早于3秒，给用户感知系统在认真工作）
