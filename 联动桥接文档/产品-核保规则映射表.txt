# 产品类型 → 核保规则映射表

## 用途
当用户选择/配置了一款产品后进行核保时，Hub 根据产品类型自动匹配对应的核保策略。本文档供 LLM 检索使用。

---

## 一、重疾险产品 → 核保规则

### 等待期与核保严格度
- 等待期 90天：健康核保 **标准严格**。任何轻微异常（如高血压I级、轻度脂肪肝）都需审慎评估
- 等待期 180天：健康核保 **可适度宽松**。轻微异常可由精算加费消化，无需直接延期/拒保
- 等待期 365天：通常为带病投保产品，核保最宽松，但费率极高

### 特色条款影响核保
- 含「轻症豁免」：核保时额外关注轻症高发风险因素
  - 轻度脑中风风险 → 重点关注血压、血脂
  - 轻微心梗风险 → 重点关注BMI、吸烟
  - 原位癌风险 → 重点关注家族肿瘤史
- 含「重症多次赔付」：核保时关注家族遗传病史（癌症、心脑血管）
- 含「运动达标增保额」：不额外增加核保负担

### 保额档位与核保要求
| 保额区间 | 核保等级 | 额外要求 |
|----------|----------|----------|
| 10万-50万 | 标准核保 | 无 |
| 50万-100万 | 标准+收入核查 | 年收入≥保额/10 |
| 100万-200万 | 严格核保+体检 | 必须提供近1年体检报告 |
| 200万-500万 | 最严格+面访 | 需体检+收入证明+生存调查 |
| 500万+ | 风控审批 | 需多层审批，非标体不可承保 |

## 二、医疗险产品 → 核保规则

### 核心原则
医疗险对既往症最敏感，是所有险种中健康核保最严格的。

### 关键核保因子（权重排序）
1. **既往症**（权重 40%）：任何已确诊疾病、体检异常均可能除外或拒保
2. **BMI**（权重 20%）：<18 或 >32 均需额外评估
3. **吸烟**（权重 15%）：吸烟者呼吸系统疾病风险显著提高
4. **职业**（权重 10%）：高危职业意外受伤概率高
5. **年龄**（权重 15%）：>55岁费率跳升，>65岁通常拒保

### 保证续保版 vs 普通版
- 保证续保版（如保证续保20年）：核保**最严格**，因续保期间不重新核保
- 普通版（1年期）：核保标准相对宽松，但续保需重新核保

### 常见除外决定
| 健康状况 | 核保决定 |
|----------|----------|
| 高血压I级（控制良好） | 标准体，但高血压相关疾病除外 |
| 高血压II级及以上 | 延期或拒保 |
| 甲状腺结节（TI-RADS 1-2级） | 标准体，甲状腺疾病除外 |
| 甲状腺结节（TI-RADS 3级） | 除外或延期 |
| 甲状腺结节（TI-RADS 4级+） | 拒保 |
| 脂肪肝（轻度、肝功能正常） | 标准体 |
| 脂肪肝（中重度或肝功能异常） | 加费或延期 |
| 乙肝携带（肝功能正常） | 除外肝脏疾病 |
| 乙肝（肝功能异常） | 拒保 |

## 三、定期寿险产品 → 核保规则

### 核心原则
定期寿险关注的是「短期内死亡风险」，因此职业风险和重大疾病史权重最高。

### 关键核保因子（权重排序）
1. **职业风险等级**（权重 35%）：5-6类职业可能拒保或大幅加费
2. **年龄**（权重 25%）
3. **性别**（权重 15%）：男性费率高于女性
4. **重大疾病史**（权重 15%）：癌症、心梗、脑卒中等
5. **吸烟**（权重 10%）：吸烟者费率增加50-100%

### 高保额额外核查
- 保额 >200万：核查收入负债比，保额不超过年收入×20
- 保额 >500万：需体检+财务核保

## 四、意外险产品 → 核保规则

### 核心原则
意外险健康核保最宽松（一般不查健康告知），但职业风险权重极大。

### 关键核保因子（权重排序）
1. **职业风险等级**（权重 50%）
2. **是否参与高危运动/活动**（权重 25%）
3. **年龄**（权重 15%）
4. **健康状况**（权重 10%）：仅严重健康问题才影响

### 职业风险与承保
| 职业等级 | 示例 | 承保决策 |
|----------|------|----------|
| 1-2级 | 内勤、程序员、教师 | 标准体，最低费率 |
| 3级 | 销售人员、护士 | 标准体，常规费率 |
| 4级 | 外卖骑手、出租车司机 | 加费20-50% |
| 5级 | 建筑工人、消防员 | 大幅加费100-300%或拒保 |
| 6级 | 矿工、高空作业、爆破 | 通常拒保 |

## 五、跨险种通用映射规则

### 年龄与核保
| 年龄段 | 核保标签 | 通用规则 |
|--------|----------|----------|
| 0-17岁 | 未成年人 | 保额上限限制（通常≤50万），需监护人投保 |
| 18-35岁 | 青年段 | 标准费率，健康核保最宽松 |
| 36-45岁 | 中年段 | 标准费率+，开始关注体检异常 |
| 46-55岁 | 中高年段 | 费率跳升约30-50%，多数需体检 |
| 56-65岁 | 高年段 | 费率跳升50-100%，严格核保 |
| 65岁+ | 老年段 | 多数险种拒保，仅少量专属产品可承保 |

### 性别与费率差异
- 重疾险：女性费率略低于男性（因女性更早体检，早期发现率高），差异≤10%
- 定期寿险：男性费率高于女性（因男性平均寿命短），差异可达30%
- 医疗险：基本无差异
- 意外险：基本无差异
