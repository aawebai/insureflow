# 核保决策结果 → 保单生成映射表

## 用途
当核保完成后，系统自动将核保决策转化为保单的初始状态和建议。供模块C和Hub检索使用。

---

## 一、标准体承保 → 正常保单

### 保单状态
- 承保状态：**正常承保**
- 费率类型：标准费率 × 风险系数 1.0
- 保障完整度：100%（所有保障责任均有效）
- 是否需要人工审核：否

### 自动生成的保单信息
- 生效日期：核保通过次日零时
- 等待期起算：生效日期
- 首次缴费日：生效日期前
- 宽限期：60天（长期险）/ 30天（短期险）

### 用户应收到
1. 保单生效确认
2. 保障责任清单（引用产品定义）
3. 缴费计划表
4. 理赔流程简述

### Hub自动引导话术
"核保通过！您的保单将于明日零时生效，等待期XX天。您可以通过以下方式管理保单：
- 查询保障范围：输入'我的保单保什么'
- 了解理赔流程：输入'怎么理赔'
- 续保咨询：输入'什么时候续保'"

## 二、次标准体承保 → 加费保单

### 保单状态
- 承保状态：**加费承保**
- 费率类型：标准费率 × 风险系数（1.2 - 2.5倍）
- 保障完整度：通常100%（加费但不减少保障），少数情况除外特定疾病
- 是否需要人工审核：视加费幅度（>2倍需人工复核）

### 风险系数参考
| 加费因素 | 加费幅度 |
|----------|----------|
| 高血压I级（控制良好） | +20-30% |
| BMI 30-35 | +20-40% |
| 轻度吸烟（<10支/天） | +15-25% |
| 职业3-4级 | +20-50% |
| 甲状腺结节（除外承保） | 标准费率（但除外） |
| 脂肪肝（肝功能异常） | +30-50% |
| 多重风险叠加 | 各项系数相乘，上限2.5-3.0 |

### 用户应收到
1. 加费承保说明（为什么加费）
2. 标准费率对比表（让用户理解加费幅度）
3. 保障完整度确认（是否所有责任都保留）
4. 改善建议（如减重可申请降费、戒烟可降费等）

### Hub自动引导话术
"您的核保结果为【加费承保】。因为XXX原因，您的年缴保费比标准费率高出XX%。但保障范围完整不变。
改善后可以申请重新核保降费：如戒烟满1年、BMI降至28以下等。"

## 三、延期体 → 暂不承保

### 保单状态
- 承保状态：**暂不承保**
- 延期类型：固定延期（如延期6个月复查）/ 条件延期（如手术后重新评估）
- 保障完整度：N/A（暂不承保）
- 是否需要人工审核：是

### 常见延期原因及期限
| 原因 | 建议延期时间 | 延期后核保条件 |
|------|-------------|---------------|
| 近期手术（<6个月） | 术后6个月 | 提供术后病理+复查报告 |
| 体检异常待复查 | 1-3个月 | 提供复查正常报告可标体 |
| 妊娠期（>28周） | 产后3个月 | 提供产后体检报告 |
| BMI ≥ 40 | 6-12个月 | 需降至35以下并提供体脂报告 |
| 新发高血压（未控制） | 3-6个月 | 血压稳定控制在<140/90 |
| 新发糖尿病（未稳定） | 6个月 | 糖化血红蛋白<7.0% |

### 用户应收到
1. 延期原因（具体到哪个健康指标）
2. 建议检查/治疗项目
3. 预计可重新核保的时间
4. 期间的风险提示建议（如先买意外险过渡）

### Hub自动引导话术
"您目前的情况建议延期XX个月后再投保。原因是XXX。在此之前您可以：
- 先配置意外险（健康核保宽松，通常可标体）
- 按医生建议控制XX指标
- XX个月后携带复查报告重新核保，大概率可标准体承保"

## 四、拒保体 → 替代方案

### 保单状态
- 承保状态：**拒绝承保**
- 拒保类型：永久拒保（严重疾病）/ 险种拒保（该险种不承保，其他可）
- 保障完整度：N/A
- 是否需要人工审核：是

### 拒保后的替代推荐矩阵
| 拒保险种 | 推荐替代1 | 推荐替代2 | 推荐替代3 |
|----------|----------|----------|----------|
| 重疾险 | 防癌险（仅保癌症，核保宽松） | 意外险+医疗险组合 | 惠民保（无健康告知） |
| 医疗险 | 惠民保（政府指导，无健康告知） | 特定疾病医疗险 | 意外险 |
| 定期寿险 | 意外险（含身故责任） | 带身故责任的重疾险 | — |
| 意外险 | 极少数情况会拒保 | 惠民保 | — |

### 常见拒保原因
| 原因 | 严重程度 | 其他险种可能 |
|------|----------|-------------|
| 恶性肿瘤病史 | 极高 | 仅惠民保/特定防癌险 |
| 心梗/脑卒中史 | 极高 | 意外险可能承保 |
| 肾功能衰竭 | 极高 | 仅惠民保 |
| 肝硬化 | 极高 | 仅惠民保 |
| 精神分裂症 | 非常高 | 意外险可能承保 |
| 艾滋病/HIV | 极特殊 | 极少数专属产品 |

### 用户应收到
1. 拒保原因详细解释
2. 替代方案推荐（具体到险种和原因）
3. 惠民保等兜底选项
4. 不推荐的做法警告（如不告知健康状况带病投保）

### Hub自动引导话术
"很遗憾，基于您目前的情况，该险种暂时无法承保。但我为您找到了替代方案：
方案1：防癌险（仅关注癌症风险，核保更宽松，可标准体承保）
方案2：惠民保（无健康告知，年保费仅100-200元）
方案3：意外险（不查健康，提供意外身故和医疗保障）
需要了解其中某个方案吗？"
